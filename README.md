# Mini Store

Prueba técnica — tienda en línea con Next.js 14, TypeScript y Tailwind.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
npm run dev
```

Abrir en `http://localhost:3000`

## Cuentas de prueba

- **Admin:** admin@ministore.com / admin123
- **Cliente:** customer@ministore.com / customer123

## Deploy en Vercel

1. Subir el repo a GitHub
2. Entrar a [vercel.com](https://vercel.com) e importar el repositorio
3. No requiere variables de entorno
4. Click en Deploy
